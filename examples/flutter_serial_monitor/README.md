# Platform Serial Example App

Professional Flutter example for the `platform_serial` package.

## Included Capabilities

- serial port discovery and selection
- complete serial parameter configuration
- connection open/close lifecycle management
- read/write serial terminal with color-coded entries
- light/dark theming (Material 3)
- splash screen with app logo
- about window and license page
- multilingual localization:
  - English, French, Portuguese, Spanish, Italian, German, Dutch
  - Russian, Greek, Turkish
  - Arabic, Hebrew
  - Chinese, Japanese, Korean

## Run

```bash
flutter pub get
flutter run
```

## Test

```bash
flutter test
```
